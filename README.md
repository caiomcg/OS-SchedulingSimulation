A C++ simulation of FCFS(First-Come, First-Served), SJF(Short Job First) and RR(Round Robin) scheduling algorithms.

## Requirements ##

* C++ Compiler.

## Build Instructions ##

* Clone the project.
* Move to the debug folder.
* Invoke the makefile.

```
git clone https://github.com/caiomcg/OS-SchedulingSimulation.git
cd debug
make all
```
